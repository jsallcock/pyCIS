class pycis.model.interferometer.UniaxialCrystal(orientation, thickness, cut_angle, material='a-BBO', material_source=None, contrast=1)[source]¶

Plane-parallel uniaxial birefringent crystal.

Parameters:
  • orientation (float) – Orientation of component fast axis in radians, relative to the x-axis.
  • thickness (float) – thickness in m.
  • cut_angle (float) – Angle in radians between crystal optic axis and front face.
  • material (str) – Crystal material.
  • material_source (str) – Source of Sellmeier coefficients describing dispersion in the crystal. If blank, the default material source specified in pycis.model.dispersion
  • contrast (float) – An arbitrary contrast degradation factor for the retarder, independent of ray path. Value between 0 and 1. Simulates the effect of real crystal imperfections.

class pycis.model.interferometer.SavartPlate(orientation, thickness, material='a-BBO', material_source=None, contrast=1, mode='francon')[source]¶

Savart plate.

Parameters:
  • orientation (float) – Orientation of component fast axis in radians, relative to the x-axis.
  • thickness (float) – thickness in m.
  • material (str) – Crystal material.
  • material_source (str) – Source of Sellmeier coefficients describing dispersion in the crystal. If blank, the default material source specified in pycis.model.dispersion
  • contrast (float) – An arbitrary contrast degradation factor for the retarder, independent of ray path. Value between 0 and 1. Simulates the effect of real crystal imperfections.
  • mode (str) – Determines how impartedd delay is calculated: ‘francon’ (approx.) or ‘veiras’ (exact).
