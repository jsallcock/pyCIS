class pycis.model.interferometer.UniaxialCrystal(orientation, thickness, cut_angle, material='a-BBO', material_source=None, contrast=1)¶

Uniaxial birefringent crystal.

get_delay(wavelength, inc_angle, azim_angle, n_e=None, n_o=None)¶

calculate phase delay (in rad) due to uniaxial crystal.

Veiras defines optical path difference as OPL_o - OPL_e ie. +ve phase indicates a delayed extraordinary ray

Parameters

  • wavelength – wavelength in m.

  • inc_angle – ray incidence angle in radians.

  • azim_angle – ray azimuthal angle in radians.

  • n_e – manually set extraordinary refractive index (for fitting)

  • n_o – manually set ordinary refractive index (for fitting)

Returns

delay in radians.

get_fringe_frequency(wavelength, focal_length)¶

Parameters

Returns

class pycis.model.interferometer.SavartPlate(orientation, thickness, material='a-BBO', material_source=None, contrast=1, mode='francon')¶

Savart plate

get_delay(wavelength, inc_angle, azim_angle, n_e=None, n_o=None)¶

calculate phase delay (in rad) due to Savart plate.

Vectorised. If inc_angle and azim_angle are arrays, they must have the same dimensions. source: Lei Wu, Chunmin Zhang, and Baochang Zhao. “Analysis of the lateral displacement and optical path difference in wide-field-of-view polarization interference imaging spectrometer”. In: Optics Communications 273.1 (2007), pp. 67–73. issn: 00304018. doi: 10.1016/j.optcom.2006.12.034.

Parameters

  • wavelength (float or array-like) – wavelength [ m ]

  • inc_angle (float or array-like) – ray incidence angle [ rad ]

  • azim_angle (float or array-like) – ray azimuthal angle [ rad ]

  • n_e (float) – manually set extraordinary refractive index (for fitting)

  • n_o (float) – manually set ordinary refractive index (for fitting)

Returns

phase [ rad ]
